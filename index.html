Linux终端中用于目录导航和管理简单的文本文件的一些命令摘要

常用命令中 CMD 命令和 Linux 命令行的差异

pwd命令———–查看当前工作目录，输出到屏幕上的是绝对路径

  1. 绝对路径是以根目录（“/”）为起点的完整路径，以所要到的目录为终点
  2. 相对路径是以当前目录（“.”）为起点的完整路径，以所要到的目录为终点


ls——————-列出当前目录所有文件和文件夹

mkdir 目录名——————在当前目录新建文件夹

cd 目录名———————进入到指定的目录

使用 cd 目录名 进到指定目录，如果指定的“目录名”是


. 表示当前目录

.. 表示当前目录的上一级目录

- 表示上一次所在目录

～ 表示当前用户的 home 目录（即刚 login 时所在的目录）

python的简单语法

  1. 运算符有五种：+ - * / %
  2. 运算顺序：先乘除 再加减 括号最优先
  3. 变量，就是一个标签，由非数字开头的字母、数字、下划线组成，它的内容可以是数值、字符串、列表、元组和字典。

数值，就是数字。
字符串，就是用一对儿双引号、或单引号引起来的内容，只要被引号引起来，就是字符串了。
